## 2.4 Novel UX Patterns

**Established Patterns (Use As-Is):**
- ROI Dashboard: SaaS analytics dashboards are well-understood
- Before/After Comparison: "Since deployment" is standard
- Progressive Disclosure: Summary first, drill-down optional
