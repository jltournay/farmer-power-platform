# Leaf Type Coaching Cards (TBK Integration)

## Strategic Purpose

The TBK grading model provides **actionable diagnostic data** through leaf type classification. Coaching cards translate each Secondary leaf type into visual, understandable improvement guidance that can be delivered through multiple channels.

## Coaching Card Design System

Each leaf type that causes Secondary grading has a dedicated coaching card:

---

## Coaching Card Delivery Strategy

| Channel            | Format                           | Best For                |
|--------------------|----------------------------------|-------------------------|
| **Factory Kiosk**  | Large printed poster with photos | All farmers at delivery |
| **SMS**            | Brief tip (part of grading SMS)  | Immediate feedback      |
| **Voice IVR**      | Spoken coaching (2-3 key actions) | Low-literacy farmers    |
| **WhatsApp**       | Image card + brief text          | Smartphone users        |
| **Extension Visit** | Printed handout + demonstration  | High-touch coaching     |

## Coaching Card Integration Points

**1. SMS (Automatic):**
```
Tatizo: majani 3+ (coarse plucking)
Kesho: chuma 2 tu + bud
```

**2. Dashboard (Joseph's View):**
- Click "View Coaching Card" button on farmer detail page
- Shows relevant card for farmer's top issue
- Option to print or send via WhatsApp

**3. Voice IVR (Farmer Call):**
- System plays coaching audio for detected top issue
- "Jinsi ya kuboresha: Chuma majani mawili tu na bud..."

**4. Extension Officer App:**
- Receives coaching card PDF when assigned a farmer
- Uses as training tool during visit
